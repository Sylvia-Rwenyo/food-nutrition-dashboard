# Food Nutrition Dashboard

The **Food Nutrition Dashboard** is a web application that visualizes nutritional data from food datasets. It processes CSV files containing food nutrition information using a Python script and displays the results in an interactive React frontend styled with Tailwind CSS. The app includes a filterable table, summary insights (e.g., top foods by nutrients), and caloric range groupings. The project is built with Vite and deployed on GitHub Pages.

## ðŸš€ Features

- **Data Processing**  
  Merges multiple CSV files, cleans data, and generates JSON files with:
  - Summary statistics
  - Top nutrient-rich foods
  - Nutrient correlations

- **Interactive Dashboard**  
  - Filterable table of food items with nutritional metrics (Calories, Fat, Protein, Carbohydrates, Nutrition Density)  
  - Summary card showing top foods by nutrients and nutrient-dense options  
  - Caloric range groupings for easy categorization

## âš™ï¸ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/sylvia-rwenyo/food-nutrition-dashboard.git
cd food-nutrition-dashboard
```

### 2. Install Frontend Dependencies

```bash
npm install
```

Installs React, Vite, Tailwind CSS, etc.

### 3. Install Python Dependencies

```bash
pip install pandas
```

### 4. Prepare CSV Files

Place your CSV files (`food-data-group1.csv`, `food-data-group2.csv`, etc.) in the project root.  
Ensure each file includes the columns:

- `food`
- `Caloric Value`
- `Fat`
- `Protein`
- `Carbohydrates`
- `Nutrition Density`

### 5. Run the Python Script

```bash
python food_data_analysis.py
```

Generates:

- `analyses/food_data.json`: Main dataset for the table  
- `analyses/summary_analyses.json`: Analysis results  
- `cleaned_food_data.csv`: Cleaned CSV (optional)

Copy the JSON files to the `public/analyses/` directory:

```bash
mkdir public/analyses
copy analyses/*.json public/analyses/
```

### 6. Run the Development Server

```bash
npm run dev
```

Visit: [http://localhost:5173](http://localhost:5173)

Check that:

- The table displays food data  
- The summary card appears  
- Caloric groups are listed  
- The filter input works

### 7. Build for Production

```bash
npm run build
```

Preview locally:

```bash
npx serve dist
```
